# VBA_INITIAL

Fichier Excel Visual Basic réalisé dans le cadre de mon travail chez Engineering Conception Maintenance.

Ce fichier est réalisé selon l'approche POO.

Il vise à l'édition des differents "Ordres de Mission" pdf en fonction des données inscrites dans la feuille "Entrée" du fichier Excel.
La feuille "Entrée" peut elle aussi être éditée en passant par la feuille "Input".
Ce fichier tient en compte des indemnités de kilometrages affectables ou pas à l'ordre de mission.
